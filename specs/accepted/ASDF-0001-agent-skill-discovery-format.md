# ASDF‑0001
Agent Skill Discovery Format

Defines the root-level discovery file (`ASDF.md`) that provides repository context to agents.

Agents should search for context files in the following order:

1. ASDF.md
2. AGENTS.md
3. CLAUDE.md
4. .cursorrules
5. CONTRIBUTING.md
