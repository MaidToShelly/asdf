# ASDF Repository Context

This repository defines the **Agent Skill Discovery Format (ASDF)** specification.

## Purpose

ASDF provides a unified standard for:

- discovering agent context in repositories
- defining reusable skills
- composing deterministic strategies
- safely executing agent workflows

## Structure

/specs – formal specification documents  
/examples – example skills and strategies (future)  
/registry – future skill registry

## Do Not

- store secrets in skill definitions
- allow undeclared capabilities
- bypass capability approval checks
